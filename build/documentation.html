                  Installation instructions
                  SRI GOKULAM HOSPITAL installation instructions

                  SRI GOKULAM HOSPITAL is an advanced, responsive admin template built with React, Redux, Next.js and Tailwind CSS.

                  It includes 2 different layouts, 2 background, 2 navbar and left sidebar color styles, 100+ components in every layout and lots of widgets and custom made reusable components to help you kickstart your next React project or application.

                  To install, first make sure node, npm and yarn are installed in your system. If they are not installed, you can get node and npm here and yarn here.

                  To check if they are installed correctly, run the following commands, which should print at least the following versions.

                  1. node -v v10.16.1
                  2. npm -v 6.14.3
                  3. yarn -v 1.22.4

                  SRI GOKULAM HOSPITAL is 100% ready to use. Just unzip the included files and run the following commands:

                  1. yarn install to install all the required dependencies
                  2. yarn dev to run the development server
                  3. Open http://localhost:3000

                  To build and run a production ready bundle, run the following commands:

                  1. yarn build
                  2. yarn start to preview the generated files
                  3. To optimize the css file sizes, we recommend you uncomment the purge option available in the tailwind.config.js file at line 3. This will decrease the css file size by around 90%.

                  File and folder code structure

                  SRI GOKULAM HOSPITAL uses the default Next.js file and folder code structure. For more information on the project and to familiarize yourself with the file and folder structure used in a Next.js application, please go to the Next.js website.

                  Naming conventions
                  File and folder definitions and naming conventions used in the template

                  All pages are located in the src/pages folder. There are no restrictions on where you put your pages, so feel free to modify this naming convention.

                  All components and widgets are located in thesrc/components folder.

                  All layouts and structural views are located in thesrc/layouts folder.

                  CSS files are located in thesrc/css folder.

                  Customization
                  Frequently asked questions
                  1. How do I open the template?
                    First, make sure the create-next-app package is installed. For more information go to this url. Then you need to run yarn install to install the template followed by yarn dev to start the development server
                  2. I need an empty page to start developing. Where can I find one?
                    Go to the src/pages/empty-page.js file
                  3. How do I add a new page and/or route?
                    Routes are handled automatically by Next.js. To add a new page just add it to the src/pages folder.
                  4. How do I set a default and/or global page layout?
                    Global layouts are defined in the src/reducers/config.js file. Layouts for every route are defined in the src/layouts/index.js file.
                  5. Where can I find the colors available in the template?
                    Colors are defined in the tailwind.config.js file in the colors section, starting in line 38. The same colors are copied to the src/functions/colors.js which makes them available for javascript components such as charts and maps.
                  6. How do I add a css file to the template?
                    CSS files are imported into the template by the src/pages/_app.js file. To add a new file just import it to this file.
                  7. How do I set the page background, left sidebar, navbar, logo right sidebar colors?
                    Palette colors are defined in the css file of each component. For example, for the Navbar component, palette colors can be changed in the css/components/navbar.css file.
                  8. How do I change the colors of other elements and components, such as tables and dropdowns?
                    Colors are defined for most elements in the css/components folder and can either be set there globally or be modified using the Tailwind CSS background, text and border color modifiers. All available modifiers are listed in the https://tailwindcss.com/docs/text-color/#app Tailwind CSS website.
                  9. I have a question, but it is not listed here, what can I do?
                    Send us an email to support@mobifica.com or contact us through our Theme Forest profile and we'll do our best to try to help you
                  10. I would like to see a new plugin installed or functionality developed in the template, is it possible?
                    We are always trying to improve and adding new functionalities and plugins to the template, so every request is welcomed and will be evaluated
                  11. I found a bug, what can I do?
                    Please let us know and we'll do our best to fix it as soon as possible
                  12. Are updates free?
                    Yes, they are and always will be for our Themeforest customers
